AI Tutor for Kids turns your child's educational websites into interactive learning experiences:

1. **Navigate** to any educational website (FLVS, Khan Academy, Wikipedia, etc.)
2. **Click** the graduation cap button that appears

## üöÄ Quick Start Guide

### Step 1: Install the Extension

**Option A: From Chrome Web Store** (Coming Soon)
- Visit the Chrome Web Store listing
- Click "Add to Chrome"
- Done!

**Option B: Manual Installation** (For Now)
1. Download this repository from [GitHub](https://github.com/JonSamprand/ai-tutor-for-kids)
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the downloaded folder
6. The extension icon will appear in your toolbar!

### Step 2: Get Your OpenAI API Key

**Watch our tutorial video:** [Link to your tutorial video]

**Quick steps:**
1. Go to [platform.openai.com](https://platform.openai.com)
2. Sign up or log in
3. Navigate to "API Keys" in the left menu
4. Click "Create new secret key"
5. Copy the key (starts with `sk-proj-...`)

**Cost:** Typically $0.01-0.05 per tutoring session (very affordable!)

### Step 3: Configure the Extension

1. Click the extension icon in your Chrome toolbar
2. Paste your API key
3. Keep the default settings (or customize for your child's age)
4. Click "Save & Apply Settings"

### Step 4: Start Tutoring!

1. Go to any educational webpage
2. Look for the floating üéì button (bottom right corner)
3. Click it to start the tutor
4. Hold the microphone button and speak your question
5. The AI will respond with helpful guidance!

**Pro tip:** Press and hold the SPACEBAR to talk (easier than clicking!)

## üí° Tips for Best Results

### For Parents
1. **Start supervised:** Watch the first few sessions to see how it works
2. **Customize the prompt:** Adjust the teaching style for your child's needs
3. **Set time limits:** Keep sessions to 10-15 minutes for younger children
4. **Review conversations:** Scroll through the chat to see what was discussed
5. **Monitor API costs:** Check your OpenAI dashboard monthly

### For Students
1. **Speak clearly:** Talk at a normal pace in a quiet space
2. **Be specific:** Ask clear questions about what you don't understand
3. **Think first:** Try to answer the AI's guiding questions
4. **Take your time:** The AI is patient - no rushing needed
5. **Ask follow-ups:** If you're still confused, ask for clarification

### For Better Performance
1. **Good internet:** Stable connection helps with voice processing
2. **Quiet environment:** Reduces speech recognition errors
3. **Clear microphone:** Use a good-quality mic if available
4. **Simple pages:** Text-heavy content works better than complex interactive sites

## üêõ Troubleshooting

### "Please configure your API key"
- Click the extension icon
- Make sure you've pasted your key correctly
- Key should start with `sk-proj-` for OpenAI
- Click "Save & Apply Settings"

### Microphone not working
- Check browser permissions (click the üîí lock icon in address bar)
- Allow microphone access for the website
- Try refreshing the page
- Test your microphone in other apps

### AI responses are slow
- Check your internet connection
- Make sure you have API credits remaining
- Try during off-peak hours
- Consider using a faster AI model

### Speech recognition not working
- Make sure you're using Chrome or Edge (Safari/Firefox not supported)
- Speak clearly and at a normal pace
- Reduce background noise
- Check browser microphone permissions

### "Extension context invalidated"
- This happens when the extension updates
- Simply reload the webpage
- The error is harmless
