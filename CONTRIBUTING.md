## ğŸš€ Getting Started

### Prerequisites
- Chrome or Edge browser
- OpenAI API key (for testing)
- Basic knowledge of JavaScript and Chrome Extensions

### Setup
1. Fork this repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ai-tutor-for-kids.git`
3. Load the extension in Chrome:
   - Go to `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the cloned folder
4. Make your changes
5. Test thoroughly

## ğŸ“ Pull Request Guidelines

### Before Submitting
- [ ] Test your changes with real educational content
- [ ] Ensure no console errors in normal use
- [ ] Update documentation if needed
- [ ] Follow existing code style
- [ ] Keep changes focused (one feature/fix per PR)

### Code Style
- Use clear, descriptive variable names
- Add comments for complex logic
- Keep functions small and focused
- Use ES6+ features when appropriate
- Follow existing patterns in the codebase

### Commit Messages
Use clear, descriptive commit messages:
```
Good: "Add keyboard shortcut for voice activation"
Bad: "Update content.js"
```
